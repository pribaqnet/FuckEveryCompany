## Installation:
```
    git clone https://github.com/pribaqnet/FuckEveryCompany.git
    pip install tweepy
    crontab -e
```
In the cron config, add the following line:
```
30 * * * * cd /dir/to/the/project &&  /usr/bin/python3 main.py
```
